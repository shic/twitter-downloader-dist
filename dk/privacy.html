Fortrolighedspolitik

Efter vores Servicevilkår, opsummerer dette dokument praksis vedrørende indsamling af personlige oplysninger i forbindelse med brug af vores hjemmeside og de tjenester, vi tilbyder. Dette inkluderer oplysninger, der leveres under brugen af tjenesterne.

Vi kræver tydeligt, at alle brugere af denne service skal være 18 år eller ældre, eller opfylde den lovmæssige myndighedsalder for deres jurisdiktion, hvis den er højere. Den højeste aldersgrænse gælder. Personer under denne alder må ikke bruge denne service eksplicit. Vi indsamler eller anmoder ikke bevidst om personlige oplysninger fra personer under denne alder.

Indsamling af information

Brug af tjenesten. Ved adgang til tjenester, brug af søgefunktioner, konvertering af filer eller download kan ikke-personlige oplysninger (såsom IP-adresse, oprindelsesland osv.) indsamles på enheden eller computeren. Dette kan bruges til registrering af information, indsamling af trafikaggregeringsdata og mulig brug af information og/eller indhold uden tilladelse.

Brugerdata. Vi kan indsamle data om brug af vores tjenester og statistiske oplysninger såsom søgninger, browsing og downloads.

Uploaded indhold. Alt indhold uploadet, downloadet eller adgang til via tjenesten kan muligvis indsamles af os.

Kommunikation. Vi kan gemme al kommunikation med os.

Cookies. Vi bruger cookies til at identificere browser-sessioner unikt under brug af tjenester. Dette kan omfatte både sessionscookies og vedvarende cookies.

Brug af data
Dine oplysninger kan bruges til at tilbyde bestemte funktioner i tjenesten og til at skabe en tilpasset oplevelse. Det kan også bruges til at vedligeholde, betjene og forbedre tjenestefunktioner.

Vi kan bruge cookies, web beacons og andre teknologier til at gemme information, levere tilpasset indhold og information ved fremtidige besøg, overvåge effekten af tjenesten og indsamle aggregateringsmålinger såsom samlet antal besøgende og sidevisninger (herunder sporing af besøgende fra affiliate-sider). Disse teknologier kan også bruges til at levere målrettet reklame baseret på dit land eller andre personlige data.

Dine personlige data kan kombineres med oplysninger fra andre brugere til formål som markedsføring og kampagner.

Dine oplysninger kan bruges til administration af kampagner, konkurrencer, undersøgelser og andre aktiviteter og funktioner.

Oplysningsoverdragelse
I visse situationer kan vi være forpligtet til at videregive specifikke data for at overholde lovgivningsmæssige krav eller for at håndhæve vores Servicevilkår og andre aftaler. Vi kan også dele oplysninger med andre virksomheder, organisationer, lovhåndhævende myndigheder eller regeringer for at beskytte vores egne rettigheder, ejendom eller sikkerhed, samt beskytte dine og offentlighedens sikkerhed, selv uden eksplicit angivelse i vores Servicevilkår, herunder deling af oplysninger med andre virksomheder, organisationer, lovhåndhævende myndigheder eller regeringer for at forebygge eller straffe ulovlig aktivitet.

Hvis der mistænkes eller rapporteres om upload, adgang eller overførsel af ulovligt eller unøjagtigt materiale via tjenesten, kan vi uden forudgående varsel videregive alle relevante oplysninger, herunder til relevante myndigheder og rettighedshavere.

Andet

Vi tager kommercielt rimelige fysiske, administrative og tekniske sikkerhedsforanstaltninger for at beskytte dine oplysninger, men vi kan ikke garantere fuldstændig sikkerhed ved dataoverførsel via internettet. Derfor er transmission af oplysninger eller indhold til os på egen risiko og ansvar.
